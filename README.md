# 114 Spring - Recurrent Neural Network and Transformer

## Course Overview

This course provides an in-depth introduction to two major AI model families:

- Recurrent Neural Networks (RNNs)
- Transformers

Both are widely used for sequential data modeling and natural language processing.

## Course Content

### Recurrent Neural Networks

The course begins with the basic structure and operating principles of RNNs, with emphasis on how recurrent architectures learn temporal dependencies in sequence data.

It also covers two important RNN variants:

- Long Short-Term Memory (LSTM)
- Gated Recurrent Unit (GRU)

These models are introduced as solutions to the long-term dependency problems found in standard RNNs, with discussion of their strengths and tradeoffs.

### Transformers

The second major topic is the Transformer architecture. The course introduces its main design principles, including:

- self-attention
- multi-head attention

It also discusses BERT and related Transformer-based models, along with their applications in:

- natural language processing
- image processing
- computer vision

### Modern Large Language Models

The course further examines more recent Transformer-based AI systems, especially large language models such as the GPT family.

Related topics include:

- continual domain-adaptive pretraining
- low-rank fine-tuning
- retrieval-augmented generation
- other modern adaptation and deployment techniques

## Learning Goal

Through practical examples and programming assignments, the course aims to help students understand how these neural network models work and how to apply them to real-world problems.
